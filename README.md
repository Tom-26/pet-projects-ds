# NumPy vs Numba
В работе сравниваются Python-списки и массивы NumPy, а также три реализации умножения матриц: ручной алгоритм на вложенных циклах, JIT-ускорение через Numba и векторизованное умножение `np.dot`.

Результаты показывают, что чистый Python сильно проигрывает по скорости, Numba ускоряет циклы в сотни раз, а NumPy оказывается самым быстрым вариантом для матричных операций.

# Droplet CV

Левитирующие кластеры микрокапель можно рассматривать как бесконтейнерные микрореакторы и удобную модель для изучения аэрозольных процессов, поэтому здесь важна автоматическая оценка числа капель и их оптического отклика во времени. Это полезно для задач на стыке микрофлюидики, биореакторов и оптических сенсоров, где динамика и “светимость” капель связаны со стабильностью системы и эффективностью реакций. Научный контекст: [Huaman I. A. et al. DropClust: A platform for microscopical image and video analysis applied to levitating droplet clusters //Colloids and Surfaces A: Physicochemical and Engineering Aspects. – 2025. – С. 138153.](https://www.sciencedirect.com/science/article/abs/pii/S0927775725020564).

В этом проекте я собрал первую рабочую версию CV-пайплайна для шумного видео с микрокаплями: детекция капель, подсчёт по кадрам, трекинг во времени, оценка интенсивности свечения и QC-визуализация. Сейчас baseline уже работает на вручную размеченных кадрах и показывает примерно `precision ≈ 0.916`, `recall ≈ 0.838`, `F1 ≈ 0.875`, с ошибкой подсчёта около `9` капель на размеченный кадр.

Стек: `Python`, `OpenCV`, `NumPy`, `Pandas`, `SciPy`, `scikit-image`, `napari`. Подробный README с описанием структуры проекта, артефактов и запуска лежит внутри папки с кодом проекта.
